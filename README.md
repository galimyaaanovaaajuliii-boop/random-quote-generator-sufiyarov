## 📖 Описание программы

**Random Quote Generator** — это графическое приложение для генерации случайных цитат. Программа позволяет получать вдохновляющие цитаты, добавлять новые, фильтровать по автору и теме, а также сохранять историю генераций.

### ✨ Основные возможности:

- ✅ **Генерация случайных цитат** из встроенной библиотеки
- ✅ **Добавление новых цитат** с указанием автора и темы
- ✅ **Фильтрация** по автору и теме
- ✅ **История генераций** с сохранением в JSON
- ✅ **Валидация ввода** (проверка пустых полей)
- ✅ **Библиотека цитат** с возможностью удаления
- ✅ **Автоматическое сохранение** всех данных
